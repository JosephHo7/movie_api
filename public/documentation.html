This achievement will build an API that gives the user access to information about different movies, directors, genres, etc.

Business Logic URL HTTP Method Query Params Request Body Response Body
Return list of all movies /movies GET none none return JSON object with all movie data
Return data about single movie by title /movies/:title GET :title none return JSON object with specific movie data
Return data about a genre type by name/title /movies/genre/:genreName GET :genreName none return JSON object with specific movie genre type
Return data about a director by name /movies/director/:directorName GET :directorName none return JSON object with details about a director
Allow new users to register /users POST none New user input info (username,email and password) in JSON format New user info that was added in JSON format
Allow users to update their username /user/:id POST :id Enter updated username in JSON format Updated user info (username) that was added in JSON format
Allow users to add a movie to their favorites /users/:id/movieTitle POST :id :movieTitle none Confirmation that the movie title has been added to favorites
Allow users to delete a movie from their favorites /users/:id/:movieTitle Delete :id :movieTitle none Confirmation that the movie title has been deleted favorites
Allow users to deregister /users/:id Delete :id none Confirmation that user email has been removed
