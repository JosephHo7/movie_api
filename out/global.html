Overview: This API links to a database that contains information about users and a list of movies that a user can access via the app. There are several endpoint calls that can be made that allow the user to access all movies, view details about a genre, director, or summary of a movie. The user can also add or remove certain movies to a list of favorite movies specific to each user. Finally, a user can create a new account, edit any account information, or delete an account.
